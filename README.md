# 🚨 Women Safety Route Advisor

## 📌 Overview

This project is designed to provide **safe route recommendations** for women by analyzing different factors such as location safety, risk levels, and optimized navigation paths.

## 🎯 Objective

To enhance personal safety by suggesting the **most secure travel routes** using intelligent algorithms.

## ⚙️ Tech Stack

* Python
* Flask (Backend)
* HTML, CSS (Frontend)
* Machine Learning (for safety prediction)
* Pandas, NumPy

## 📂 Project Structure

```
├── data/
├── models/
├── templates/
├── app.py
├── requirements.txt
```

## ▶️ How to Run

```
pip install -r requirements.txt
python app.py
```

## 🚀 Features

* Safe route prediction
* User-friendly interface
* Risk-aware navigation
* Real-world application use case
